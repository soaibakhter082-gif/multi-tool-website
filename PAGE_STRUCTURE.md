# PAGE STRUCTURE

## 1. Route Map

| Page | Route | Purpose |
|---|---|---|
| Home | `/` | Introduce website and highlight tools/categories |
| All Tools | `/tools` | Show all tools with search and filters |
| Category Page | `/categories/[category]` | Show tools for one category |
| Tool Page | `/tools/[slug]` | Show one tool UI and guide content |
| Blog/Guides Placeholder | `/blog` | Placeholder page for future guides |
| About Us | `/about` | Explain website mission and trust points |
| Contact Us | `/contact` | Contact form/info placeholder |
| Privacy Policy | `/privacy-policy` | Privacy and browser-processing notes |
| Terms of Use | `/terms-of-use` | Terms and usage limits |
| Disclaimer | `/disclaimer` | Liability and accuracy disclaimer |
| Sitemap | `/sitemap` | Human-readable list of all pages and tools |

## 2. Global Layout Structure
- Header with logo/name, main nav, and search entry point
- Main content area
- Footer with legal links and quick links

## 3. Home Page Sections
- Hero section with clear value message
- Category quick links
- Featured tools section
- "How this site works" short section
- Privacy trust note

## 4. All Tools Page Sections
- Page title and short intro
- Tool search bar
- Category filter chips/buttons
- Tool card grid
- Empty state message when no result

## 5. Category Page Sections
- Category title and intro
- Tool count
- Tool cards for that category
- Related categories section

## 6. Individual Tool Page Sections
- Tool title
- Short description
- Input area
- Output area
- Action buttons
- Copy button
- Reset/Clear button
- How to use
- Example
- FAQ
- Related tools

## 7. Legal and Informational Pages
- About: project story, values, commitment to privacy
- Contact: contact method placeholder + response expectation
- Privacy: what data is processed and where
- Terms: acceptable use and limitations
- Disclaimer: accuracy and responsibility boundaries

## 8. Sitemap Rules
- Build sitemap links from central registry and static route list
- Include Home, All Tools, all categories, all tool pages, legal pages
- Keep URLs clean and readable

## 9. Registry-Driven Areas
These pages/sections must read from central tool registry:
- Home featured tools
- All tools listing
- Category listings
- Related tools
- Sitemap tools section
