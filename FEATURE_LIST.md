## 3. MVP Content Features
- Home page highlights categories and top tools
- All Tools page with search and filters
- Category pages with tool cards
- Individual tool pages with help content
- Related tools section on tool pages
- FAQ section where useful
- Blog/Guides placeholder
- Legal pages and sitemap page

## 4. Browser-First Processing Plan
- Text and developer tools run fully in browser
- Calculator tools run fully in browser
- Most image and PDF tools run in browser using client-side libraries
- Show a clear privacy note for browser-only tools

## 5. Future Features (After Version 1)

- Better sharing cards (OG images per tool)
